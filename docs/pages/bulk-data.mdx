# Bulk Data Files

Download comprehensive CSV files for case law, people, and financial disclosures for large-scale data analysis and research.

## Available Datasets

### Case Law Data
- **Opinions**: Complete judicial opinions with metadata
- **Clusters**: Case-level information and citations
- **Courts**: Court jurisdiction data
- **Dockets**: Court docket information

### People Data
- **Judges**: Biographical information for judges
- **Attorneys**: Attorney information from case records
- **Financial Disclosures**: Judge financial disclosure reports

### Specialized Data
- **RECAP Documents**: Federal court document archive
- **Audio Files**: Oral argument metadata
- **Citation Network**: Citation relationships between cases

## File Formats

### CSV Format
- Standard CSV with UTF-8 encoding
- Comma-separated values
- Quoted text fields
- Header row with column names

### File Structure
```
bulk-data/
├── opinions/

```

## Usage Examples

### Python

```python
import pandas as pd

# Load opinions data
opinions = pd.read_csv('opinions.csv')

# Filter by date
recent_opinions = opinions[opinions['date_created'] > '2023-01-01']

# Analyze data
opinion_count = len(recent_opinions)
print(f"Total opinions: {opinion_count}")
```

### Command Line

```bash
# Download and analyze
wget https://bulk.courtlistener.com/opinions.csv
wc -l opinions.csv
head -n 5 opinions.csv
```

### Database Import

```sql
-- Import CSV to PostgreSQL
COPY opinions FROM 'opinions.csv' DELIMITER ',' CSV HEADER;
```
