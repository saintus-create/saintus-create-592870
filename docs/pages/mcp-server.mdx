# MCP Server

Access CourtListener data through the Model Context Protocol (MCP) server for AI assistant integration.

## Overview

The MCP Server provides a standardized interface for AI assistants like Claude, Cursor, and VS Code to access CourtListener data, enabling natural language queries and intelligent legal research.

## Getting Started

### Prerequisites
- AI assistant that supports MCP (Claude, Cursor, VS Code)
- CourtListener API credentials
- MCP client configuration

### Installation

Add the MCP server to your AI assistant configuration:

```json
{
  "mcpServers": {
    "courtlistener": {
      "command": "python",
      "args": ["-m", "courtlistener_mcp"],
      "env": {
        "COURTLISTENER_API_KEY": "your-api-key"
      }
    }
  }
}
```

## Available Tools

### Search Cases
Search for cases using natural language queries.

```python
# AI assistant can call
courtlistener.search_cases(
    query="contract breach damages",
    jurisdiction="federal",
    limit=10
)
```

### Get Opinion
Retrieve specific opinion details.

```python
courtlistener.get_opinion(opinion_id=123456)
```

### Lookup Citation
Verify legal citations.

```python
courtlistener.lookup_citation("123 F.3d 456")
```

### Get Docket
Access docket information.

```python
courtlistener.get_docket(docket_id=789012)
```

## Configuration

### Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `COURTLISTENER_API_KEY` | Yes | Your CourtListener API key |
| `COURTLISTENER_API_URL` | No | Custom API URL (default: official) |

### Advanced Configuration

```json
{
  "mcpServers": {
    "courtlistener": {
      "command": "python",
      "args": ["-m", "courtlistener_mcp"],
      "env": {
        "COURTLISTENER_API_KEY": "your-api-key",
        "COURTLISTENER_API_URL": "https://custom.api.url",
        "LOG_LEVEL": "debug"
      }
    }
  }
}
```
