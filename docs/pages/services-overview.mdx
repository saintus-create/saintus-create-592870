# Services Overview

This documentation provides comprehensive access to multiple legal APIs and data services, giving you everything you need to integrate legal data into your applications.

## Available APIs

### CourtListener API
- **Case Law**: Access millions of judicial opinions and case clusters
- **Search**: Full-text search across the entire database
- **PACER & RECAP**: Federal court docket information
- **Citation Lookup**: Verify legal citations

### Caselaw Access Project API
- **6.7M+ Cases**: Harvard's comprehensive case law database
- **360 Years of History**: From 1658 to 2018
- **Jurisdiction Filtering**: Filter by court and jurisdiction
- **Full Case Text**: Access complete case documents

### Congress.gov API
- **Legislative Data**: Bills, amendments, and member information
- **Committee Data**: Access committee assignments and hearings
- **Bill Tracking**: Monitor legislation status and history
- **Congressional Records**: Access official congressional documents

### Vaquill AI API
- **AI Legal Q&A**: Ask legal questions in plain language
- **Semantic Search**: AI-powered case law search
- **Statute Lookup**: Access US Code, CFR, and state codes
- **Contract Analysis**: AI-powered contract review

## Data Services

- **Database Replication**: Direct PostgreSQL database access
- **Bulk Data Files**: CSV exports for large datasets
- **Webhooks**: Real-time event notifications
- **MCP Server**: AI assistant integration

## Getting Started

1. Choose the API that best fits your use case
2. Review the authentication requirements
3. Explore the API reference documentation
4. Start integrating with your application
